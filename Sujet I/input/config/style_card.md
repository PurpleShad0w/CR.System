# Build4Use – Audit Report Style Card (FR)

## 1) Voice & stance
- **Professional, neutral, audit tone**. No chatty phrasing.
- Use **third person** or **impersonal** constructions.
  - Prefer: "Le site…", "L’analyse montre…", "Il ressort que…"
  - Avoid: "tu/ton/ta", "je peux", "voici", emojis.
- Do **not** invent facts, quantities, equipment, brands, dates.

## 2) How to turn notes into report text
- Always move from **raw note** → **auditor wording**:
  - Add *minimal context* (why it matters) when the evidence supports it.
  - Merge duplicated lines into a **single synthesized bullet**.
- Prefer **cause / impact / condition** structure when possible:
  - "Constat : …" → "Constat : … (impact exploitation/énergie si explicitement indiqué)"

## 3) Bullet style
- 4–8 bullets per slide.
- Each bullet <= 170 characters.
- Start with an **actionable noun** or **clear statement**:
  - "Comptage : …", "Supervision : …", "Historisation : …", "Alarmes : …"
- Avoid repeating "Le décret…" on multiple bullets: keep 1 bullet "Cadre réglementaire" then apply implicitly.

## 4) Forbidden patterns
- No emojis (👉 ✔️ ❌ ⚠️ 📊 etc.).
- No tutoiement (tu, ton, ta, tes).
- No conversational wrappers: "Oui", "Je peux…", "Ton plan…", "Je te réponds…".
- No quotes unless it is a short official reference.

## 5) Micro-templates
### Contexte général (intro slide)
- "Le présent rapport synthétise les constats disponibles pour le site : {SITE}."
- "La mission s’inscrit dans le cadre {CADRE} (décret BACS / ISO 52120-1)."
- "Objectif : qualifier l’existant et situer le niveau de performance attendu."
- "Les informations manquantes sont listées en points à confirmer."

### État des lieux (thème)
- "Comptage : {FAIT} (limites : {LIMITES})."
- "Supervision : {FAIT}."
- "Historisation : {FAIT} (durée/qualité si présent)."
- "Alarmes : {FAIT}."

### Scoring (actuel/projeté)
- "Méthode : ISO 52120-1, critère le moins-disant, sans pondération (si applicable)."
- "Classe atteinte : {CLASSE} – principaux écarts : {TOP_3}."
- "Cible : {CLASSE_CIBLE} – conditions / hypothèses : {HYPOTHESES}."

## 6) Captions (légendes photo)
- 2–6 words, concrete, location + equipment.
  - Examples: "CTA aile centre RDJ", "Chaufferie aile château".
